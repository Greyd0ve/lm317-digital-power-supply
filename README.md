# 基于 LM317 的数控直流电压源设计

## 项目简介
本项目基于 **STM32F103C8T6 + MCP4725 DAC + 运放调理 + LM317 线性稳压** 实现一款数控直流电压源。  
系统支持数字设定、步进调节、OLED 实时显示、软件校准与闭环微调，可实现输出电压的稳定可调。

## 主要功能
- 输出电压数字设定
- 两档步进调节（0.1V / 1.0V）
- OLED 显示设定值、实际值、步进值、DAC 值
- 输出电压/电流实时采样
- 软件校准提高输出精度
- 过压、过流保护逻辑

## 技术指标
- 输出电压范围：0 ~ 12V
- 输出电压精度：±0.1V 以内
- 最大输出电流：1.0A（设计目标）
- 控制核心：STM32F103C8T6
- 调压方式：MCP4725 DAC + 运放调理 + LM317

## 实测结果
经软件拟合校准后：
- 最大绝对误差：0.054V
- 平均绝对误差：0.0366V
- 满足题目要求的 ±0.1V 精度指标

## 系统结构
系统采用以下总体方案：

- 单片机控制模块
- DAC 数控调节模块
- 运放调理模块
- LM317 稳压输出模块
- 电压采样模块
- 电流采样模块
- 按键输入模块
- OLED 显示模块
- 软件保护模块
